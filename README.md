# create-joji-app

A CLI tool to quickly scaffold React + Vite + Tailwind CSS projects with Jolyssa's preferred setup.

## Installation
```bash
npm link
```

## Usage
```bash
create-joji-app [project-name]
```

Or just run it and follow the prompts:
```bash
create-joji-app
```
